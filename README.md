# Teslo API

1. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```

2. Cambiar las variables de entorno

3. Levantar la base de datos
```
docker compose up -d db
```

4. Levantar el proyecto en modo desarrollo
```
yarn start:dev
```
